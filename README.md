# World Bank Education Dashboard

**Interactive Power BI dashboard analyzing global education indicators**

This project presents an interactive Power BI dashboard built using World Bank Education Statistics (EdStats) data.

The dashboard analyzes global education patterns including education access, gender parity, literacy outcomes, and government investment in education.

The goal of this project was to transform complex education data into a clear and interactive visual analysis.

---

## Project Goal

The objective of this project was to design a structured and interactive dashboard that helps users explore global education trends.

The dashboard enables:

• Comparison of countries  
• Exploration of long-term trends  
• Understanding of gender parity  
• Analysis of education investment

---

## Dashboard Overview

The dashboard is organized into three analytical sections.

### Page 1 – Education Access

Focuses on global access to education.

Includes:

• Global enrolment distribution map  
• Education access trends over time  
• Country-level enrolment comparison  
• Key performance indicators (KPIs)

---

### Page 2 – Gender & Literacy

Focuses on gender equality in education.

Includes:

• Gender parity distribution  
• Literacy gender parity trends  
• Global parity comparison  
• Key performance indicators (KPIs)

---

### Page 3 – Policy & Economy

Focuses on education investment and policy indicators.

Includes:

• Education spending distribution  
• Education spending trends  
• Top countries by education investment  
• Key performance indicators (KPIs)

---

## Dataset

**Source**

World Bank Education Statistics (EdStats)

**Indicators Used**

• Adjusted net enrolment rate (Lower Secondary)  
• Enrolment Gender Parity Index (GPI)  
• Youth Literacy Gender Parity Index (GPI)  
• Government expenditure on education (% of GDP)  
• GDP PPP (reference indicator)
